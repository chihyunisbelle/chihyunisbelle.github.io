Project: Great Timing

Click to explore the code

a time and task tracking app designed to help users to manage time with interactive and fun visualization, a solo project experimenting with Python, CSS and Django


Breaking down the features:

entering the site, visitors/users need signup/login first to start using the app

once logged in, the app will bring you to the overview of all the 'projects' you have setup, with the pie chart and percentage of the tasks complete/incomplete

by pressing on a certain project, the app will bring you to the list showing all the tasks that needed to accomplish for the project. In the list, by pressing on certain task, you can edit or delete the task base on the need.

by clicking on the top left 'My tasks' icon, you can see all the tasks for all the projects showing in a timeline format, order by the earliest deadline date. Tasks that are done (not deleted) will be moved down to bottom of the timeline.

the 'Search Porject' function in the middle of the top band, allows users to search projects of certain user(company) by their company name

'company' function lists out all the users signup to the app and the number of projects they have, by clicking on certain user, this will lead to the user's detail page which lists out all the project the user set up
